# Приложение "Английские словосочетания"

Лабораторная работа №1 (Вариант 20): Приложение для анализа и хранения английских словосочетаний.

## Описание

Приложение позволяет:
- Загружать PDF-файлы с английским текстом
- Извлекать слова и словосочетания из текста с использованием NLTK
- Хранить информацию о словах и их связях в базе данных PostgreSQL
- Искать слова и просматривать связанные с ними словосочетания
- Добавлять и удалять слова и словосочетания

## Требования

- Python 3.7+
- PostgreSQL
- Библиотеки Python (устанавливаются через pip):
  - PyQt5
  - nltk
  - psycopg2-binary
  - pymupdf
  - python-dotenv

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

   - Установите PostgreSQL на вашу систему
   - Создайте базу данных для приложения
   - Обновите файл `config.py` с правильными параметрами подключения

3. Запустите SQL-скрипт для создания структуры базы данных:
```sql
-- Выполните команды из файла model/database.sql в вашей базе данных
```

## Настройка

1. Скопируйте `config_example.py` в `config.py`:
```bash
cp config_example.py config.py
```

2. Отредактируйте `config.py`, указав правильные параметры подключения к базе данных.

## Запуск приложения

```bash
python main.py
```

## Использование

1. Запустите приложение
2. Используйте меню "Файл" → "Загрузить английский текст (PDF)" для загрузки PDF-файла
3. Приложение автоматически извлечет слова и словосочетания из текста
4. Используйте панель поиска для поиска слов
5. Просматривайте словосочетания для выбранных слов в правой части интерфейса
6. Используйте кнопки "Add New Word", "Add Collocation Link" и "Delete Entry" для управления данными

## Особенности

- Автоматическое определение частей речи
- Извлечение словосочетаний с использованием NLTK
- Прогресс-бар при обработке PDF-файлов
- Валидация ввода данных
- Обработка ошибок
