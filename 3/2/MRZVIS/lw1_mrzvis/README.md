# Лабораторная работа №1 — МРЗвИС

**Вариант:** алгоритм вычисления произведения пары 4-разрядных чисел
умножением с младших разрядов со сдвигом множимого влево.

## Состав
- `src/` — исходники на Python:
  - `binary_number.py` — двоичное число фиксированной разрядности;
  - `pipeline.py` — арифметический конвейер;
  - `report_html.py` — построение HTML-отчёта;
  - `plots.py` — построение графиков S(n), E(n), S(r), E(r);
  - `main.py` — точка входа с CLI;
  - `trace_table.py` — трассировка в JSON;
  - `input.json` — пример входных пар.
- `report.tex`, `report.pdf` — отчёт (pdfLaTeX);
- `*.png` — графики ускорения и эффективности.

## Запуск

```bash
cd src
python main.py --n 5 --t 1 --r 4 --open          # HTML-отчёт
python plots.py                                  # графики
```

### Параметры
- `--n` — число пар;
- `--r` — разрядность (по умолчанию 4);
- `--t` — время счёта на этапе t_i в у.в.е.;
- `--input` — путь к JSON-файлу с парами;
- `--out` — путь для HTML-отчёта;
- `--open` — открыть отчёт в браузере.

## Сборка отчёта
```bash
pdflatex report.tex && pdflatex report.tex
```
